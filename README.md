# Miesto Renginiai – Web programa

Ši web programa leidžia miesto gyventojams ir svečiams:

- Rasti informaciją apie mieste vykstančius renginius  
- Skelbti savo renginius  
- Įvertinti renginius  

Programa susideda iš **dviejų dalių**:

1. **Administracinė** – NodeJS backend su autentifikacija  
2. **Viešoji sritis** – React SPA su Bootstrap dizainu  

---

##  Technologijos

- **Frontend:** React, Bootstrap, JavaScript, HTML5, CSS  
- **Backend:** NodeJS, Express  
- **Duomenų bazė:** MySQL (per Docker)  
- **Versijavimas:** Git  
- **Lokali aplinka:** Windows 11, VS Code
